World Cup Bot

Open-source Polymarket LP bot for FIFA 2026 advance markets — with Kalshi cross-venue scan + optional auto-exec

World Cup Bot is an MIT-licensed Python stack for liquidity provision on FIFA World Cup 2026 advance to knockout markets on Polymarket. Shadow-first deployment, calendar cancel window before kickoff, versioned JSONL ledger, streak + portfolio risk gates (on by default), and Polymarket vs Kalshi cross-venue gap scan with paper ledger (Phase A/B) and optional gated dual-leg auto-exec (Phase C, off by default).

  • Conviction YAML + Gamma mids at runtime (no hardcoded prices)
  • Streak sizing + portfolio risk gates on by default (risk-status)
  • Shadow mode default (DRY_RUN=true) with phased SHADOW.md checklist
  • Cross-venue scanner for PM / Kalshi cohort pairs — paper by default; Phase C on non-US egress
  • Optional Module 8 in-play shock recovery — paper-first, off by default
