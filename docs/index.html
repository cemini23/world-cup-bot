World Cup Bot

Open-source Polymarket LP bot for FIFA 2026 advance markets — with Kalshi cross-venue alerts

World Cup Bot is an MIT-licensed Python stack for liquidity provision on FIFA World Cup 2026 advance to knockout markets on Polymarket. It includes shadow-first deployment, a calendar cancel window before kickoff, versioned JSONL ledger, and optional Polymarket vs Kalshi gap alerts (alert-only, no auto execution).

  • Conviction YAML + Gamma mids at runtime (no hardcoded prices)
  • Shadow mode default (DRY_RUN=true) with phased SHADOW.md checklist
  • Cross-venue scanner for PM / Kalshi cohort pairs
